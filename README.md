# SeeU Education: WeChat Mini-Program Full-Stack Development

## Project Overview
**SeeU Education** is a leading education consulting agency dedicated to bridging the gap between college students and professional opportunities. This project involved the end-to-end development and deployment of a web-based **WeChat Mini-Program**. 

The application serves as a centralized virtual portal for members to access career resources, participate in community events, and manage their professional development journey. Prior to this phase, the program lacked critical interactive features, user authentication, and data-driven engagement tools.

## Key Features Implemented
* **User Authentication & Security:** Implemented a secure member-only login system to gate exclusive consulting resources.
* **Activity Management:** Developed a lookup system for incoming career activities and workshops.
* **Community Voting System:** Built a democratic voting feature allowing members to influence which future activities or guest speakers the agency hosts.
* **Multimedia Integration:** Enhanced the UI/UX by integrating dynamic image loading and visual assets for a more engaging user experience.
* **Collections & Rewards:** Designed a database-backed system for users to save resources (Collections) and track engagement (Reward Points).

## Tech Stack
* **Frontend:** Vue.js (within the WeChat Mini-Program framework)
* **Backend:** Java Spring Boot (RESTful API architecture)
* **Database:** MySQL
* **DevOps & Deployment:** Docker, AWS (EC2/RDS), Nginx (Reverse Proxy), SSL/TLS Certification

## My Responsibilities & Contributions
As a **Full-Stack Developer**, I was responsible for bridging the gap between data architecture and user-facing functionality.

### Backend & Database Engineering
* **Database Schema Design:** Architected and managed the **MySQL** database, specifically designing schemas for the reward points system and user activity tracking.
* **API Development:** Built robust **Java Spring Boot** RESTful endpoints to handle complex business logic, including the voting algorithms and secure authentication flows.

### Full-Stack Integration
* **Frontend Connectivity:** Authored frontend JavaScript API wrappers and service layers to ensure seamless communication between the Vue.js interface and the Spring Boot backend.
* **Collaborative Support:** Provided documentation and integration support for other frontend developers to utilize the developed backend services.

### DevOps & Infrastructure
* **Dockerization:** Containerized the Spring Boot application and MySQL environment for consistent deployment across staging and production.
* **Cloud Deployment:** Managed the end-to-end deployment on **AWS**.
* **Network Security:** Configured **Nginx** as a reverse proxy and implemented **SSL/TLS certificates** to enable secure HTTPS communication, a requirement for WeChat Mini-Program production environments.

## Results & Impact
* **Successful Launch:** Successfully transitioned the program from a prototype to a live production environment.
* **User Satisfaction:** Post-launch surveys indicated that users found the new features (Activity Lookup, Voting, and Rewards) significantly more useful than the previous iteration.
* **KPI Growth:** The implementation of these features resulted in a **satisfaction rate increase of over 70%** among the agency’s student members.
